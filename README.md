# Loan Analytics Dashboard

A comprehensive business intelligence solution for analyzing loan performance, assessing portfolio risk, and driving strategic decision-making.

## Project Description

This dashboard provides stakeholders with real-time visibility into loan portfolio performance, including metrics on repayment behavior, risk distribution, and growth trends.

## Key Features

- Real-time portfolio performance tracking
- Loan performance analysis and trend detection
- Geographic distribution and regional analysis
- Financial KPI monitoring (funded amount, received amount, interest rates, DTI metrics)
- Risk management and default tracking
- Interactive dashboards for different user levels

## Data Structure

The project includes loan data with the following information:
- Loan identifiers and issue dates
- Loan amounts and repayment amounts
- Interest rates and debt-to-income ratios
- Loan grades and purposes
- Loan status and quality classifications
